# Reloj PWA

Pequeña Progressive Web App que muestra la hora actual del sistema y se actualiza automáticamente cada segundo.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Service Worker
- Web App Manifest

## Funcionalidades

- Visualización de la hora actual en formato HH:MM:SS
- Actualización automática cada segundo
- Instalación como aplicación móvil
- Soporte offline mediante Service Worker

## Ejecución

Abrir el archivo:

index.html

o publicar en GitHub Pages.
